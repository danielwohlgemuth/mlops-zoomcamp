# MLOps Zoomcamp

This repository is a follow-along of the MLOps course from DataTalks.Club.

## [Intro](/01-intro/README.md)

Covered:
- Setup of Anaconda to run Jupyter Notebooks.
- Use notebooks and taxi trip data to go through machine learning concepts.
- Describe an MLOps pipeline to show how the process could be automated.
- Present the MLOps maturity model to help understand the current situation and guide toward more robust operation.
